#### 4. Finding link images.late.htb
I notice the link `images.late.htb` and click on it. I am unable to access it so I add `images.late.htb` into `/etc/hosts`

``` bash
sudo echo "10.10.11.156 images.late.htb" >> /etc/hosts
```
